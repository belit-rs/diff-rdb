## Installation

#### Leiningen/Boot
`[belit/diff-rdb "0.1.0"]`

#### Clojure CLI/deps.edn
`belit/diff-rdb {:mvn/version "0.1.0"}`

#### Gradle
`compile 'belit:diff-rdb:0.1.0'`

#### Maven
```xml
<dependency>
  <groupId>belit</groupId>
  <artifactId>diff-rdb</artifactId>
  <version>0.1.0</version>
</dependency>
```
